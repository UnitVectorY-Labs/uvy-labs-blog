Welcome to the **UnitVectorY Labs Blog** — the home for updates, deep dives, and project spotlights from [UnitVectorY Labs](https://github.com/UnitVectorY-Labs), a collection of open source tools and utilities focused on cloud infrastructure, security, and developer tooling.

## What Is UnitVectorY Labs?

UnitVectorY Labs is an open source organization building practical tools that solve real problems in the cloud-native and security space. Our work spans Google Cloud Platform (GCP) utilities, security cataloging, and lightweight infrastructure helpers — all built in the open so the community can use, contribute, and learn from them.

## Highlighted Projects

Here's a look at some of the things we've been building:

### [gcp-iam-catalog](https://github.com/UnitVectorY-Labs/gcp-iam-catalog)

Navigating GCP IAM can be overwhelming — hundreds of roles, thousands of permissions. The `gcp-iam-catalog` is a comprehensive, browseable catalog of GCP IAM roles and permissions that makes it easy to answer the question: *"Which roles include this specific permission?"*

### [gcpmetadataexplorer](https://github.com/UnitVectorY-Labs/gcpmetadataexplorer)

A web-based interface for browsing and inspecting the GCP metadata server. Perfect for understanding what metadata is available to your workloads, built in Go for efficiency.

### [mcp-graphql-forge](https://github.com/UnitVectorY-Labs/mcp-graphql-forge)

A lightweight, configuration-driven MCP server that exposes curated GraphQL queries as modular tools, enabling intentional API interactions from your AI agents. Built in Go.

### [jwks-catalog](https://github.com/UnitVectorY-Labs/jwks-catalog)

A catalog of JWKS (JSON Web Key Set) endpoints for popular websites. A handy reference for anyone building authentication flows that need to verify JWTs from third-party identity providers.

### [dns-caa-catalog](https://github.com/UnitVectorY-Labs/dns-caa-catalog)

Tracks CAA (Certification Authority Authorization) DNS configurations across top websites, surfacing insights on certificate issuance policies. A useful resource for security researchers and those hardening their own DNS configurations.

### [ServiceAuthCentral](https://github.com/UnitVectorY-Labs/ServiceAuthCentral)

Simplify microservice security with ServiceAuthCentral: centralized, open source authorization in the cloud — minus the shared secrets. Built in Java, this tackles the tricky problem of service-to-service authentication at scale.
